# Simple Pong

A two-player Pong game built with Python and Turtle graphics.

## Requirements

- Python 3.x
- Windows (uses `winsound` for audio)

## How to Run

```bash
cd Pong
python Pong.py
```

## Controls

| Player | Move Up | Move Down |
|--------|---------|-----------|
| Player A (left) | `W` | `S` |
| Player B (right) | `Up Arrow` | `Down Arrow` |

Hold the key down to move continuously.

## Gameplay

- First player to let the ball pass their paddle gives the other player a point.
- Score is displayed at the top of the screen.
- The ball speeds up slightly each time it bounces off a paddle.
